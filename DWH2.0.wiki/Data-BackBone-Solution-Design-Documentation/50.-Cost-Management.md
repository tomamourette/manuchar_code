In this section, the different cost components linked to the Data BackBone project in Microsoft Fabric, and rules for allocation of those costs, will be described.

Important to note is that billing & cost management can not directly done in the Microsoft Fabric portal.
As it stands now, Microsoft Fabric capacities are deployed, scaled and managed using Microsoft Azure.
Therefore, the billing & cost management is also done in Azure, linked to the resource group, subscription and management group of the Azure-based capacity resource(s).
